Comment utilisé l'application pour changer de vlan :

I - Installation :
	Placer le dossier Vlan_Changer dans "C:\Program Files\"
	Assurer vous que la case "Executer en tant qu'administrateur" est bien active
	Créer un raccourci et le placer dans "shell:startup"
	Modifier les fichiers "NameNetworkCard.csv" et "VlanListe.csv" par rapport a l'environnement
	Attention : Pour chaque modification redemmarré le logiciel

2 - Utilisation :
	Quand le logiciel est lancé, il apparait une icone de notification a coté de l'horolge dans la fleche qui affiche les icone cachées (vous pouvez la déplacer a coté de l'horologe pour un acces rapide
	Un survole sur l'icon indique dans quel vlan notre carte réseau se trouve
	Un clic droit fait apparaitre un petit menu où l'on peux séléctionné son vlan rapidement
	Un double clic ouvre une petite application où l'on peux également séléctionné son vlan rapidement
	Une fois le vlan choisi une notification apparait pour annoncer que le vlan a bien été changer
