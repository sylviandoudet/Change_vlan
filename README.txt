Comment utilisé l'application pour changer de vlan :

I - Installation :
	Placer le dossier Vlan_Changer dans "C:\Program Files\"
	Assurer vous que la case "Exécuter en tant qu'administrateur" est bien active


2 - Utilisation :
	Quand le logiciel est lancé, il apparait une icone de notification a coté de l'horloge dans la flèche qui affiche les icone cachées (vous pouvez la déplacer a coté de l'horloge pour un accès rapide
	Un clic droit ouvre un menu qui permet de changer de vlan rapidement ainsi que modifier les paramètres de l'application
		Paramètres -> Cartes réseau, permet de changer la cartes réseau qui va être modifié 
		Paramètres -> Listes vlans, permet de modifier la liste des vlans
		VLAN, permet de modifier rapidement le vlan de sa cartes réseau
	Un survole sur l'icon indique dans quel vlan notre carte réseau se trouve
	Un double clic ouvre une petite application où l'on peux modifié la liste des vlan rapidement
	Une fois le vlan choisi une notification apparait pour annoncer que le vlan a bien été changer
